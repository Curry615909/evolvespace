【农业】国庆期间农业板块市场行情更新(王明琦)
→ 国庆期间猪价持续上涨，一方面假期部分集团场有减量，另一方面散户供应情绪不强
→ 从白条走货看，节后屠宰量逐步回落至国庆备货前水平，消费仍有待观察
→ 随着节日效应结束，预计规模场恢复正常出栏，需求稍有下降，叠加政策指导，预计猪价持续冲高的势头或有所放缓
→ 但在养殖端对后市预期乐观的背景下，猪价或仍有支撑，11月前整体或维持震荡偏强走势

【化工】能源开采与化工四季度相对看好投资主线(邓胜)

【机械】特斯拉人形机器人点评(常义乐)
→ 尽管特斯拉人形机器人的量产节奏以及目前可以实现的功能还不及市场预期
→ 但是这款产品走在正确的道路上，底层算法构架、可量产性依旧是迄今为止我们看到的最好产品没有之一

【REITs】深度解析中国REITs估值定价(王一冰)
→ 中国REITs定价以NAV为锚定基础，收益法为底层资产估值方法
→ 可用P/NAV评估上市后产权类REITs，P/FFO评估经营权类REITs，同时以现金分派率确定REITs二级市场价格合理区间
→ 产权类REITs估值处于合理区间上沿附近，未来底层资产估值增值将带动REITs增值
→ 建议将经营权类REITs看做类固收工具，推荐关注同时满足：现金分派收益高且底层资产受疫情冲击较小的标的

【机械】海外制裁对于中国半导体设备行业影响(韩非)
→ 相关专家留去问题冲击整体半导体设备行业高端制造路线
→ 长江存储、合肥长鑫等芯片制造厂商未来扩产节奏不确定导致设备厂商前景不乐观
